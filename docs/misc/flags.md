# Flags

List of flag names used by [Flag()](../funcs/flag.md).



```ahk
"is_in_air"         Checks if entity is airborne.
"is_sneaking"       Checks if entity is sneaking.
"is_sprinting"      Checks if entity is sprinting.
"is_swimming"       Checks if entity is swimming.
"is_on_fire"        Checks if entity is on fire.
"is_baby"           Checks if entity is a baby.


; {item} = any vanilla item id string. Example: "is_holding_grass_block_mainhand"

"is_holding_{item}"             Checks if entity is holding an item.
"is_holding_{item}_mainhand"    Checks if entity is holding an item in the mainhand.
"is_holding_{item}_offhand"     Checks if entity is holding an item in the offhand.
```
